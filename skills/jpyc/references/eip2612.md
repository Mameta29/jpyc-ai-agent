# EIP-2612: Permit (Gasless Approve)

## permit

```solidity
function permit(
    address owner,     // Token owner (authorizer, signs the message)
    address spender,   // Address to approve
    uint256 value,     // Allowance amount
    uint256 deadline,  // Unix timestamp expiry (must be >= block.timestamp)
    uint8 v,
    bytes32 r,
    bytes32 s
) external
```

- Caller can be **anyone**
- Sets `allowance[owner][spender] = value` if signature is valid
- Reverts if: paused, `owner` or `spender` blocklisted, expired, invalid signature

## nonces

```solidity
function nonces(address owner) external view returns (uint256)
```

Returns the current sequential nonce for `owner`. Increments by 1 after each successful `permit` call.

**Important**: EIP-2612 nonces are sequential `uint256`, unlike EIP-3009 which uses random `bytes32`.

## Signature Construction (TypeScript / viem)

```typescript
import { parseEther } from 'viem'

const JPYC_ADDRESS = '0xE7C3D8C9a439feDe00D2600032D5dB0Be71C3c29' as const

// 1. Get current nonce
const nonce = await contract.read.nonces([ownerAddress])

// 2. Build typed data
const deadline = BigInt(Math.floor(Date.now() / 1000) + 3600)

const typedData = {
  domain: {
    name: 'JPY Coin',
    version: '1',
    chainId: 1,
    verifyingContract: JPYC_ADDRESS,
  },
  types: {
    Permit: [
      { name: 'owner',    type: 'address' },
      { name: 'spender',  type: 'address' },
      { name: 'value',    type: 'uint256' },
      { name: 'nonce',    type: 'uint256' },
      { name: 'deadline', type: 'uint256' },
    ],
  },
  primaryType: 'Permit' as const,
  message: {
    owner:    ownerAddress,
    spender:  spenderAddress,
    value:    parseEther('1000'),
    nonce,
    deadline,
  },
}

// 3. Sign (owner must sign)
const signature = await walletClient.signTypedData(typedData)

// 4. Decode v, r, s from signature
const r = `0x${signature.slice(2, 66)}` as `0x${string}`
const s = `0x${signature.slice(66, 130)}` as `0x${string}`
const v = parseInt(signature.slice(130, 132), 16)

// 5. Submit (anyone can relay)
await contract.write.permit([
  ownerAddress, spenderAddress, parseEther('1000'),
  deadline, v, r, s
])
```

## Comparison with transferWithAuthorization

| Feature | permit (EIP-2612) | transferWithAuthorization (EIP-3009) |
|---|---|---|
| Purpose | Set allowance off-chain | Transfer tokens off-chain |
| Nonce type | Sequential `uint256` | Random `bytes32` |
| Nonce getter | `nonces(owner)` | `authorizationState(from, nonce)` |
| Time control | `deadline` only | `validAfter` + `validBefore` |
| Caller restriction | Anyone | Anyone (TWA) / payee only (RWA) |
